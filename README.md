
## Installation  
**Windows** 
1. Download ffmpeg   
2. Add bin folder to PATH (i.e. `...\ffmpeg-master-latest-win64-gpl\bin`)  
3. Make sure ffmpeg is installed by running `ffmpeg` in cmd prompt, restart IDE if needed

Change `file = "10.mp3" # file to be processed` in `main.py` to your audio file.
